Adds a local pick-up otion to your wp e-commerce cart by GetShopped.org

== Description ==

Adds a local pick-up otion to your wp e-commerce cart by GetShopped.org. Shipping can be set to free or any amount.  

== Installation ==

Install plugin and activate.  Local Pick-up option will be added to your wp ecommerce shipping options.
