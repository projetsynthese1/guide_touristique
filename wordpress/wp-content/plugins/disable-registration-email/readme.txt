== Description ==
= Disable Registration Email for WordPress =
Turns off the automatic "New user registration on your blog" email messages to the admin email address. Useful for large sites.

= Usage =
* Nothing to configure. Simply activate and the emails stop coming!

== Changelog ==
= 0.9.0 =
* Initial release


== Installation ==
1. Download the *Disable Registration Email* plugin .zip file.
1. Unzip the archive
1. Upload the `disable-registration-email` folder to `wp-content/plugins/`.
1. Go to the *Plugins* tab.
1. Activate *Disable Registration Email*.
